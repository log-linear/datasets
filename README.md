# datasets

Miscellaneous datasets and scripts used to generate/update them.

## Current datasets

- dfw_historical_weather - Historical NOAA weather data for the DFW area since 2013
- us_holidays - US holiday dates since 2013 from holidata.net

## Script dependencies

- R _(>= 4.1.0)_
    - dplyr
    - here
    - rnoaa
    - tidyr
    - readr
- PowerShell
- git (obviously)
